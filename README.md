# 🌐 imnse.com  

The official digital flagship repository for the IMNSE ecosystem, serving as the central hub for public visibility, developer onboarding, global growth, and user education.  

> [!NOTE]  
> **Repository Scope:** This project focuses entirely on production-grade user acquisition and documentation layers. Core application runtimes, AI engines, and hardware frameworks are decoupled into their respective standalone repositories.  

---

## 🏗️ Web Architecture & Directory Structure  

* 🏛️ `/public-site`  
  * The primary marketing gateway for `imnse.com`, showcasing the core value proposition, corporate identity, enterprise solutions, and system status updates.  
* 📄 `/docs`  
  * Developer-first technical documentation, API specs, onboarding tutorials, and deployment manuals generated via high-performance static site components.  
* ✍️ `/blog`  
  * Content hub for engineering updates, product feature announcements, industry insights, and SEO-optimized organic growth publications.  
* 🎯 `/landing-pages`  
  * Conversion-optimized marketing interfaces, rapid A/B testing variations, and tracking pipelines configured specifically for Google Ads and organic campaign funnels.

## 🚀 Quick Start for Web Developers & Growth Marketers  

### Prerequisites  
* Install **Node.js LTS** environment on your local development workstation.  
* Ensure your account possesses authorized write access tokens to the centralized marketing Content Management System (CMS).  

### Local Engine Initialization  
```bash
# Clone the foundational marketing and public asset repository  
git clone https://github.com  
cd imnse.com  

# Download package dependencies and build all workspaces  
npm install  
npm run build  

# Spin up the local development preview server  
npm run dev -- --open  
```
